# 具身智能讲解脚本（3分钟）

【0-30秒 开头】
大家好，欢迎来到“大家讲AI”。今天我讲的主题是：具身智能。
AI已经能下棋、画画、聊天，但为什么很难像人一样倒水、开门？
答案就是具身智能。它是AI从屏幕走向现实的关键。

【30秒-2分钟 主体】
一、什么是具身智能？
具身智能 = 让AI拥有身体，拥有感知、决策、行动的完整能力。
传统AI只负责思考，具身智能能和现实世界互动。

二、具身智能 vs 传统AI
1. 学习方式：传统AI从数据学习；具身智能在环境中学习
2. 能力范围：传统AI单一任务；具身智能完成复杂任务
3. 物理理解：传统AI没有身体；具身智能懂重力、距离、力度

三、核心技术
多模态感知、导航规划、机器人控制、大模型任务规划、仿真训练

四、应用场景
家庭机器人、医疗康复、工业协作、人形机器人、自动驾驶
